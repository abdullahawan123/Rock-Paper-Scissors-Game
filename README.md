# Rock‑Paper‑Scissors Game

A classic Rock‑Paper‑Scissors game built using **Flutter** and **Dart**. This project demonstrates simple game logic with interactive UI elements, allowing users to play against a computer opponent. The game runs on Android, iOS, and web platforms.

## Features

- Classic Rock‑Paper‑Scissors game rules
- Play against computer with random decisions
- Interactive buttons and responsive UI
- Built with Flutter for cross‑platform support

## Tech Stack

- Flutter
- Dart
